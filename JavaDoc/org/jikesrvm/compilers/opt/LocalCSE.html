Class LocalCSE

  • java.lang.Object
    • org.jikesrvm.compilers.opt.driver.CompilerPhase
      • org.jikesrvm.compilers.opt.LocalCSE

  • public class LocalCSE
extends CompilerPhase
    Perform local common-subexpression elimination for a factored basic block.
    • Note: this module also performs scalar replacement of loads
    • Note: this module also performs elimination of redundant nullchecks, boundchecks, and zero checks.
    Algorithm: Muchnick pp.379-385
