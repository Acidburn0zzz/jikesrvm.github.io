org.jikesrvm.compilers.opt.ir.operand

Class TypeOperand

  • java.lang.Object
    • org.jikesrvm.compilers.opt.ir.operand.Operand
      • org.jikesrvm.compilers.opt.ir.operand.TypeOperand

  • public final class TypeOperand
extends Operand
    A TypeOperand represents a type. Used in checkcast, instanceof, new, etc. It will contain either a RVMType (if the type can be resolved at compile time) or a TypeReference (if the type cannot be resolved at compile time).
    See Also:
    Operand, RVMType, TypeReference
