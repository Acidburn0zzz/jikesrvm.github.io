org.jikesrvm.compilers.opt.lir2mir

Class MinimalBURS

  • java.lang.Object
    • org.jikesrvm.compilers.opt.lir2mir.BURS
      • org.jikesrvm.compilers.opt.lir2mir.MinimalBURS

  • final class MinimalBURS
extends BURS
    This class contains code for quick and dirty instruction selection by forcing each instruction to be a tree and generating the trees in the same input as the input LIR instructions. This results in poor code quality, but can be done very quickly. The intended purpose is to reduce compile time by doing quick and dirty instruction selection for infrequently executed basic blocks.
