    public abstract class BaselineGCMapIterator
extends GCMapIterator
implements BaselineConstants
    Iterator for stack frame built by the Baseline compiler.

    An Instance of this class will iterate through a particular reference map of a method returning the offsets of any references that are part of the input parameters, local variables, and java stack for the stack frame.

    • Constructor Detail

      • BaselineGCMapIterator

        public BaselineGCMapIterator(WordArray registerLocations)
        Constructor. Remember the location array for registers. This array needs to be updated with the location of any saved registers. This information is not used by this iterator but must be updated for the other types of iterators (ones for the opt compiler built frames) The locations are kept as addresses within the stack.
