org.mmtk.vm

Class Factory

  • java.lang.Object
    • org.mmtk.vm.Factory
  • Direct Known Subclasses:
    Factory

    public abstract class Factory
extends Object
    This class defines factory methods for VM-specific types which must be instantiated within MMTk. Since the concrete type is defined at build time, we leave it to a concrete vm-specific instance of this class to perform the object instantiation.
