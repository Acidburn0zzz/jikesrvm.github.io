Class ThreadQueue

  • java.lang.Object
    • org.jikesrvm.scheduler.ThreadQueue

  • public class ThreadQueue
extends Object
    An unsynchronized queue data structure for Threads. The current uses are all where there is some other lock that already protects the queue.
