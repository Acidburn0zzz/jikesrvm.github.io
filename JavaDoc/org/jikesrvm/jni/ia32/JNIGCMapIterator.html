    ArchitectureSpecific.JNIGCMapIterator

    public abstract class JNIGCMapIterator
extends GCMapIterator
implements BaselineConstants
    Iterator for stack frames inserted at the transition from Java to JNI Native C. It will report JREFs associated with the executing C frames which are in the "JREFs stack" attached to the executing Threads JNIEnvironment. It will update register location addresses for the non-volatile registers to point to the registers saved in the transition frame.
